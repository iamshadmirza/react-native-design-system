# Avatar

Avatars are used to represent a user and can contain photos or text.

# Usage

```js
import {Avatar} from 'react-native-design-system';

<Avatar
  title="MD"
  source={{uri: 'image_url_here'}}
  onPress={() => this.handlePress()}
/>;
```

# Types

There are mainly three types og Avatar: `default`, `square` and `rounded`. All props accept `boolean` value.

# Title

`title` accepts a string. Passing upto 3 character (maximum) is recommended.

<Preview>
  <Story name="title">
    <Avatar title="MD" />
  </Story>
</Preview>

# Source

`source` is same as Image component. You can pass a `required` image or a `uri` object.

<Preview>
  <Story name="source">
    <Avatar source={{uri: 'https://www.github.com/iamshadmirza.png'}} />
  </Story>
</Preview>

# Shadow

`shadow` prop can be used to customize shadow level. Allowed values are `none`, `xs`, `sm`, `md`, `lg`, `xl`.

<Preview>
  <Story name="shadow">
    <Avatar
      shadow="lg"
      source={{uri: 'https://www.github.com/iamshadmirza.png'}}
    />
  </Story>
</Preview>

# Radius

`radius` prop customises the roundness of Avatar. Allowed values are 'none', `xs`, `sm`, `md`, `lg`, `xl`, `2xl`, `3xl`, `full`.

<Preview>
  <Story name="radius">
    <Avatar
      radius="full"
      source={{uri: 'https://www.github.com/iamshadmirza.png'}}
    />
  </Story>
</Preview>

# Editable

`editable` is a boolean value which adds an edit button to the bottom. You just have to pass one `onPress` function and the `Avatar` is clickable.

<Preview>
  <Story name="editable">
    <Stack direction="horizontal">

      <Avatar
        source={{uri: 'https://www.github.com/iamshadmirza.png'}}
        editable={true}
        onPress={action('pressed')}
      />
    </Stack>
  </Story>
</Preview>
