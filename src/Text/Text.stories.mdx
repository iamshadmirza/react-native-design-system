# Text

Simple `Text` element that gives you different sizes and colors.

# Usage

```js
import {Text} from 'react-native-design-system';

<Text>Hello Design System</Text>;
```

# Scale

Pass `scale={false}` if you don't want text to scale when device default font size changes. Default is `true` and text will scale accrording to default font size.
