# ListItem

A ListItem is a wrapper around `TextInput` with commonly used styles and easy to make customization.

# Usage

```js
import {ListItem} from 'react-native-design-system';

<ListItem chevron>This is a ListItem</ListItem>;
```

# Props

<Props of={ListItem} />

# Type

There are three types of button mainly: `default`, `round`, and `outline`. You can pass boolean value `round={true}` for round button, `outline={true}` for outline button and nothing for default solid button.

<Preview>
  <Story name="type">
    <Stack>
      <ListItem>This is a ListItem</ListItem>
      <ListItem chevron>This is a ListItem</ListItem>
      <ListItem chevron subtitle="This is a subtitle">
        This is a ListItem
      </ListItem>
    </Stack>
  </Story>
</Preview>

# Chevron

Pass `chevron={true}` to show chevron at left. You can pass `chevronColor` to change its color.

<Preview>
  <Story name="chevron">
    <ListItem chevron>This is a ListItem</ListItem>
  </Story>
</Preview>

# Subtitle

Pass a string to `subtitle` prop to go below list text.

<Preview>
  <Story name="subtitle">
    <ListItem chevron subtitle="a generic subtitle">
      Hey there!
    </ListItem>
  </Story>
</Preview>

# Avatar

Pass `avatarSource` to get an avatar on left. It receives prop same as an Image component.

<Preview>
  <Story name="avatarSource">
    <ListItem
      chevron
      avatarSource={{uri: 'https://avatars.io/twitter/iamshadmirza'}}
      subtitle="a generic subtitle">
      Hey there!
    </ListItem>
  </Story>
</Preview>

# Icon

You can `leftIcon` and `rightIcon` that goes to left and right on the `ListItem`. You can customize icon with `iconStyle`.

<Preview>
  <Story name="icon">
    <ListItem
      chevron
      subtitle="@iamshadmirza"
      leftIcon={<Feather name="twitter" color="#333" size={24} />}
      rightIcon={<AntDesign name="adduser" color="#333" size={24} />}>
      Follow me on twitter
    </ListItem>
  </Story>
</Preview>

# Disabled

`disabled={true}` property restrict any kind of action on ListItem.

<Preview>
  <Story name="disabled">
    <ListItem chevron disabled subtitle="a generic subtitle">
      Hey there!
    </ListItem>
  </Story>
</Preview>

# Text Color

You can pass any of the color present inside `textColor` in theme. Same property applies to `subtitleColor`.

<Preview>
  <Story name="textColor">
    <ListItem chevron textColor="success" subtitle="a generic subtitle">
      Hey there!!
    </ListItem>
  </Story>
</Preview>

# Text Align

You can align ListItem text position by passing `auto`,`left`, `center`, `right` or `justify` as `textAlign` property.

<Preview>

        Home

        Home
      </ListItem>

  </Story>
</Preview>
