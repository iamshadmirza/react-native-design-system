# Stack

Stack is a layout container which alligns it's children vertrically (stack) and provide constant margin around them.

# Usage

```js
import {Stack, Input} from 'react-native-design-system';

<Stack size="lg">
  <Input
    placeholder="Enter Username"
    value=""
    outline
    onChangeText={text => handleUsernameChange(text)}
  />
  <Input
    placeholder="Enter Email"
    value=""
    outline
    onChangeText={text => handleEmailChange(text)}
  />
  <Input
    placeholder="Enter Password"
    value=""
    outline
    onChangeText={text => handlePasswordChange(text)}
  />
</Stack>;
```

# Direction

Give `direction="horizontal"` to make it behave like Inline. Direction can either be "vertical" or "horizontal". Default is "vertical".

# Horizontal Space

`horizontalSpace` prop adds horizontal spacing. Default is `none` but you can any of the 8 spacing values provided.
