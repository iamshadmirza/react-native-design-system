# Box

Box is a simple layout container which accepts a child and provide padding around it. The child is centered by default.

# Usage

```js
import {Box} from 'react-native-design-system';

<Box size="lg">
  <Button>Press Me</Button>
</Box>;
```

# Props

<Props of={Box} />

# Space

`space` receives values from `none`, `xs` upto `9xl`.

<Preview>
  <Story name="space">
    <Box size="lg">
      <Button>Press Me</Button>
    </Box>
  </Story>
</Preview>

# Background

You can also pass a `background` color. Color string should be present inside `colors` of theme.

<Preview>
  <Story name="background">
    <Box background="bg100" space="9xl">
      <Button>Press Me</Button>
    </Box>
  </Story>
</Preview>
