# Header
Header is just a fixed Header bar that sits on top of screen.

# Usage
```js
import { Header } from 'react-native-design-system';

<Header>
  Home
</Header>
```
> If you're using `Expo` then you have to provide top margin for StatusBar:

```js
import { Constants } from 'expo'

...
headerStyle: { marginTop: Constants.statusBarHeight },
```

You can align Header title position by passing `auto`,`left`, `center`, `right` or `justify` as `textAlign` property.

# Icons
You can pass `leftIcon` and `rightIcon` with `onLeftIconPress` and `onRightIconPress` handler. Size, type and color of icon is upto developer.

<Preview>
  <Story name="icons">
    <Header
      leftIcon={
        <EvilIcons
          name="navicon"
          size={26}
          color="white" />
      }
      rightIcon={
        <Feather
          name="settings"
          size={24}
          color="white" />
      }
      onLeftIconPress={action('left icon pressed')}
      onRightIconPress={action('right icon pressed')}
    >
      Home
    </Header>
  </Story>
</Preview>

# Bar Color
You can pass StatusBar **backgroundColor** as `barColor` and **barStyle** for StatusBar `barColor`

# Icon Styles
You can pass `iconStyle` to pass custom styling for left and right. This is handy when you have to provide some extra padding to the icons.
