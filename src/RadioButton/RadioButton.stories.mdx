# RadioButton

A RadioButton is a graphical control element that allows the user to choose only one of a predefined set of mutually exclusive options.

# Usage

```js
import {RadioButton, RadioItem} from 'react-native-design-system';

<RadioButton activeId={activeId} selectItem={id => selectItemHandler(id)}>
  <RadioItem id="a">Option A</RadioItem>
  <RadioItem id="b">Option B</RadioItem>
  <RadioItem id="c">Option C</RadioItem>
</RadioButton>;
```

# RadioButton and RadioItem

`RadioButton` and `RadioItem` are used together as compound component.

```jsx
import RadioButton, {RadioItem} from 'react-native-design-system';

<RadioButton activeId={activeId} selectItem={id => selectItemHandler(id)}>
  <RadioItem id="a">Option A</RadioItem>
  <RadioItem id="b">Option B</RadioItem>
  <RadioItem id="c">Option C</RadioItem>
</RadioButton>;
```

# activeId

Pass `activeId` which equals to one of the ids passed to children `RadioItem`. `selectItem` function received an `id` which can be used to update `activeId`. Click on **Show code** for info.
