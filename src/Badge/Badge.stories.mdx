# Badge
Badges are small components typically used to communicate a number value or indicate the status of an item to the user.

# Usage
```js
import { Badge } from 'react-native-design-system';

<Badge
  onPress={() => this.handlePress())}
>
  10000
</Badge>
```

# Types
There are 3 types of Badges mainly: `default` which is round, `sqaure` and `mini`. The prop accept boolean value.
