# Image

It's just a wrapper around RN Image component to give you more customizations like shadow and radius.

# Usage

```js
import {Image} from 'react-native-design-system';

<Image
  radius="lg"
  shadow="sm"
  source={{uri: 'https://www.github.com/iamshadmirza.png'}}
/>;
```

<Props of={Image} />
